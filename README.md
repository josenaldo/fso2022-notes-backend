## Commands

```sh
## Install
yarn install

## Run
yarn start

## Run development mode
yarn dev

## Test
yarn test

## Clean
yarn clean

## Copy UI to build folder
yarn copy:ui

## Build UI
yarn build:ui

## Deploy on fly.io
yarn deploy

## Build and deploy on fly.io
yarn deploy:full

## Logs
yarn logs:prod

## Lint
yarn lint
```
